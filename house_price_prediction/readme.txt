
Bengaluru House Price Prediction
Notebook implementing the steps you requested:

1) Download dataset (instructions included)
2) EDA / Preprocessing (pandas)
3) Visualizations (heatmap, boxplot, pairplot / swarm)
4) Train ML models: Linear Regression, SVM, GaussianNB (as classification on binned price), Decision Tree, Random Forest, KNN
5) Post-processing visual comparison of selected models

Notes:
- This is a runnable Python script / notebook. Run in a Jupyter environment or convert to .ipynb.
- You must first download the CSV from Kaggle (instructions below). The original notebook/link provided: https://www.kaggle.com/code/tariqbashir/bengaluru-house-price-predition/input
